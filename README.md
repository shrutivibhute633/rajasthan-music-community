# Rajasthan Music Community 🎵

A full-stack web application that helps promote and connect Rajasthan’s traditional music artists with the community. The platform allows artists to showcase their profiles, share media, and connect with people interested in regional music and culture.

## 🚀 Features

* Artist registration and login system
* Community page to explore Rajasthan music artists
* Artist profile management
* Media upload and sharing
* Authentication system with OTP verification
* Interactive frontend with modern UI
* Backend API for managing artists and community data

## 🛠 Tech Stack

**Frontend**

* React.js
* Vite
* CSS

**Backend**

* Python
* Django
* Django REST Framework

**Database**

* SQLite (default Django database)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/shrutivibhute633/rajasthan-music-community.git
```

### 2️⃣ Backend Setup

```
cd Tourist-api
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 3️⃣ Frontend Setup

```
cd Frontend
npm install
npm run dev
```
